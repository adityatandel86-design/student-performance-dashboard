# 📊 Final Project – Sales & Customer Intelligence Dashboard

## 📌 Project Description

This Power BI project was developed to analyze Sales, Returns, Customer Behavior, Product Performance, and Regional Performance. The report follows a Star Schema data model and uses DAX calculations to provide meaningful business insights. The dashboard is interactive, user-friendly, and designed to support decision-making.

---

# 🎯 Project Objectives

- Build a robust Star Schema Data Model.
- Create relationships between Fact and Dimension tables.
- Develop DAX Measures and Calculated Columns.
- Apply Time Intelligence (YOY, MOM, YTD).
- Build interactive dashboards using Power BI visuals.
- Implement slicers, drillthrough, bookmarks, and tooltips.
- Optimize the report for mobile devices.
- Implement Row Level Security (RLS).
